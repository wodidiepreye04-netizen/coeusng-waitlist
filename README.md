# COEUSng — Mental Health Waitlist

A private, judgment-free space to breathe, heal, and grow. Designed specifically for young Africans and Nigerians seeking premium, accessible, and warm mental health support.

This waitlist landing page is built to convey emotional warmth and safety through high-end styling, interactive micro-animations, and organic aesthetics.

---

## ✨ Key Features

1. **Premium Centered Layout**: A symmetrical, balanced desktop and mobile design that gives text and components room to breathe.
2. **Interactive Breathing Orb**: An animated, soft glass card featuring a pulsing turquoise/amber core designed to guide visitors through a calming 6-second breathing rhythm.
3. **Warm Human Illustration**: An abstract, watercolor-style silhouette illustration capturing connection, warmth, and peace.
4. **Organic Noise Texture**: A subtle, overlaying fractal noise pattern that softens the digital background, making it feel tactile and human.
5. **Emotional Typography**: High-contrast pairing of clean sans-serif text with elegant `Playfair Display` italic serifs for emotionally resonant phrases.
6. **Double-blob Mesh Background**: Ambient floating gradient shapes (teal, turquoise, and warm amber) that drift softly behind the UI elements.
7. **Mailchimp Integration**: A built-in inline waitlist form supporting Mailchimp API list subscription.

---

## 🛠️ Technology Stack

- **Framework**: React 18 & Vite
- **Styling**: Tailwind CSS v4 (with `@tailwindcss/vite` plugin)
- **Animations**: Framer Motion
- **Testing**: Vitest & React Testing Library
- **Iconography**: Custom Inline SVGs

## 🚀 Getting Started

### Prerequisites

Ensure you have **Node.js** (v18+) and **npm** installed on your system.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/wodidiepreye04-netizen/coeusng-waitlist.git
   cd coeusng-waitlist
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Development Server

Run the local development server with Hot Module Replacement (HMR):
```bash
npm run dev
```
Open **[http://localhost:5173](http://localhost:5173)** in your browser.

### Building for Production

Compile production-ready assets to the `/dist` directory:
```bash
npm run build
```
